## OS setup guide

# Base system:
- Windows 11 on 1TB disk 0
- Used `diskpart` to clear disk 1 (another 1TB drive).
- Torrent .iso, used rufus to format and flash usb disk drive (FAT32)
- Restart and boot into UEFI
- Boot iso, install onto disk 1

# Personalization
- Installed doomemacs
- Set up systemd to launch emacsclient on boot
